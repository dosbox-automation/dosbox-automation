# Naming scheme

- this is "dosbox-automation", exactly written in this convention.

# Versioning guide
- version numbering is upstream base version dash "da" release number.
  example: upstream is 0.83, our release version is 3 -> 0.83-da3

# full product name
- following the previous instructions, an example full product name is 
    dosbox-automation 0.83-da3
